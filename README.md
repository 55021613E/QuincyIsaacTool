# 夏老师以撒快捷开关使用说明

## 这是什么？

是一个《以撒的结合：忏悔》快捷开启和关闭mod、控制台，省去修改配置文件的繁琐步骤的轻量小工具。

V2.0版本更新内容：支持快速打开存档文件夹，可方便进行存档替换与迁移。

## 如何安装？

下载后双击启动运行即可。
